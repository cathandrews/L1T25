###_1. Description of the Project:_
This website showcases my passion for plants, particularly succulents, which are easy to care for and propagate. It was created to implement recently acquired skills in HTML, CSS, and JavaScript.

####_Website structure:_
- **Home Page**: A short bio and reasons why succulents are a good choice for beginners.
- **Succulent Types**: Information on different types of succulents, their growing requirements, and links to more detailed information.
- **Succulent Propagation**: A step-by-step guide with images and descriptions.
- **Contact Information**: Forms for questions, suggestions, and joining a quarterly zoom chat.
- **Save for Later**: A storage area for liked or saved content.

###_2. Installation:_
####1. **Clone the Repository**:
   git clone https://github.com/cathandrews/L1T25.git

   2. **Navigate to the Directory**
   cd L1T25

   3. **Open in Browser**
   Open the index.html file in your preferred web browser to view the website.

###_3. Usage:_
   Open the index.html file in your web browser.
   Navigate through the different sections using the menu.
   Interact with the succulent types and propagation tips by liking, saving, or hiding them.
   Use the contact form to send questions or join the quarterly zoom chat.
